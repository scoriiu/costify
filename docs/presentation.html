Ce va tine pe loc astazi
M

Introducere Manuala a Datelor

Tranzactiile sunt copiate manual din extrasele bancare in Excel. Ore de munca repetitiva in fiecare luna cu risc mare de eroare.

?

Lipsa Trasabilitatii

Nu se poate face drill-down de la un total pe categorie la tranzactia sursa. Cand cifrele nu corespund, gasirea cauzei este dureroasa.

O

Ownership Neclar

Cine este responsabil pentru revizuirea abonamentelor IT? Cine aproba cheltuielile de marketing? Responsabilitatile exista in capul oamenilor, nu in sistem.

F

Forecast-uri Invechite

Reforecast-ul necesita efort manual in Excel, asa ca se intampla rar. Pana cand un forecast este gata, realitatea s-a schimbat deja.

V

Haos de Versiuni

Multiple versiuni Excel circuland. Fara audit trail. Fara flux de aprobare. "Care fisier e cel mai recent?" se intreaba saptamanal.

D

Vizibilitate Intarziata

Datele despre solduri si cheltuieli sunt mereu cu zile sau saptamani in urma. Pana cand vedeti o problema in spreadsheet, banii sunt deja cheltuiti.

Modulele Platformei
Sapte Sisteme Integrate
Fiecare modul functioneaza independent, dar partajeaza date fara intreruperi in toata platforma.
Modulul 01

Motor de Preluare Date

Preia automat tranzactiile din FGO.ro (agregator bancar) sau direct din conturile bancare, la fiecare ora. Normalizeaza, deduplica si imbogateste fiecare inregistrare.

  • FGO.ro (agregator)
  • Banci direct
  • Import CSV/XLSX
  • Deduplicare
  • Sincronizare orara
Modulul 02

Motor de Clasificare

Motor de reguli configurabil care auto-clasifica tranzactiile dupa IBAN, furnizor, cuvinte-cheie, sume si tipare. Gestioneaza splituri pe categorii.

  • Bazat pe reguli
  • Multi-axial
  • Split tranzactii
  • Scor de incredere
  • ML-ready (Faza 2)
Modulul 03

Contabilitate & Trezorerie

Vedere in timp real a tuturor conturilor bancare, soldurilor si fluxurilor de numerar. Dashboard de trezorerie consolidat cu detalii per cont si reconciliere.

  • Sold total
  • Vedere per cont
  • Flux de numerar
  • Reconciliere
  • Multi-valuta
Modulul 04

Managementul Bugetelor

Definiti bugete anuale per verticala, entitate si categorie. Control de versiuni cu stari draft/aprobat/revizuit si fluxuri de aprobare.

  • Bugete versionate
  • Flux de aprobare
  • Import Excel
  • Blocare perioade
  • Template-uri
Modulul 05

Forecast & Optimizare

Generati prognoze folosind run-rate, sezonalitate si metode bazate pe angajamente. Comparati scenarii si identificati oportunitati de optimizare a costurilor.

  • Metode multiple
  • Scenarii
  • Optimizare
  • Tracking abonamente
  • Detectie anomalii
Modulul 06

Ownership & Acces

Fiecare categorie de cost are un owner clar. Accesul bazat pe roluri asigura ca fiecare din cei 23 de membri ai echipei vede si controleaza doar ce trebuie.

  • 6 roluri
  • Owneri per categorie
  • Notificari
  • Calendare de revizuire
  • Audit trail
Modulul 07

Integrari

Se conecteaza la FGO.ro ca agregator bancar sau direct la banci. Se integreaza cu Saga pentru trimiterea facturilor catre contabil. Flux de date bidirectional.

  • FGO.ro
  • Banci directe
  • Saga
  • Export Excel
  • API pentru BI

Planul de Implementare
De la Descoperire la Control Total
O abordare pe faze care livreaza valoare devreme si creste odata cu nevoile voastre. Timeline total: 20 saptamani.
Faza 0: Descoperire & Setup Saptamanile 1-2
Transpunem modelul Excel existent intr-o schema de baza de date validata. Evaluam capabilitatile API FGO.ro si accesul direct la banci. Importam 3-6 luni de date istorice pentru validare.
  • Validare model de date
  • Finalizare taxonomie costuri
  • Evaluare integrari
  • Setup mediu dezvoltare
  • Import date istorice
Faza 1: MVP - Platforma de Baza Saptamanile 3-8
Inlocuim fluxul de lucru Excel de baza. Import extrase bancare, clasificare tranzactii, atribuire owneri, definire bugete si raport Buget vs Forecast vs Actual cu drill-down.
  • Import extrase bancare
  • Preluare automata FGO/banci
  • Taxonomie 3 niveluri
  • Reguli de clasificare
  • Atribuire ownership
  • Definire bugete
  • Raportare BvA
  • Drill-down
  • Audit trail
  • 6 roluri utilizatori
Faza 2: Automatizare & Inteligenta Saptamanile 9-14
Clasificare avansata, scenarii de forecast, integrare Saga, tracking abonamente si sugestii de optimizare a costurilor.
  • Sincronizare banci in timp real
  • Reguli avansate
  • Split tranzactii
  • Scenarii forecast
  • Integrare Saga
  • Tracking abonamente
  • Notificari
  • Inchidere perioade
Faza 3: Raportare Avansata & Scalare Saptamanile 15-20
Constructor de rapoarte custom, clasificare asistata de ML, alerte avansate, integrare cu instrumente BI si dashboard-uri responsive mobile.
  • Rapoarte custom
  • Dashboard KPI executiv
  • Clasificare ML
  • Detectie anomalii
  • API pentru BI
  • Multi-valuta
  • Dashboard-uri mobile
