Market Analysis
The Massive Gap Nobody Fills
Romanian entrepreneurs are financially blind. They send invoices through SmartBill, give documents to the accountant, and get a P&L months later. They manage budgets in Excel — if at all.
🚫

No Real-Time Visibility

The entrepreneur learns about last month's financials next month. Decisions are based on gut feeling, not data. Bank balance is the only metric they check.

🚫

No Cost Control

"Are we on budget?" requires a phone call to the accountant. Nobody owns cost categories. Spend goes unquestioned until crisis hits.

🚫

No Forecasting

"Will we run out of cash in 3 months?" is a guess. No scenario modeling. No runway projection. Excel models rot after one use.

🚫

No Automation

Bank transactions are manually imported. Classification is manual. Reconciliation is manual. The accountant is a data entry machine instead of an advisor.

🚫

No Multi-Vertical View

Companies with multiple business lines have no consolidated view. Shared costs are split by approximation. Vertical profitability is unknown.

🚫

No Audit Trail

Who changed this number? When? Why? The answer is always "ask the accountant" or "check the old Excel." Compliance is hope-based.

Immutable Journal

Append-only transaction ledger. Corrections create new entries — existing entries are never mutated. Sequential numbering per tenant, gap-free for compliance.

🔍

Full Traceability

Any number on any dashboard can be drilled down to the source bank transaction. From KPI card → category → journal entry → raw bank data.

🛡

Tamper-Proof Audit

Every audit record includes a SHA-256 checksum. Daily background jobs verify integrity. Any mismatch triggers an alert. 10-year retention (Romanian fiscal law).

Engineering
Built for Scale, Security, and Speed

Frontend

Next.js 15, React 19, TailwindCSS. Server-rendered, responsive, dark/light mode.

Backend

Fastify (TypeScript strict). Modular services, max 100 lines per function, full audit middleware.

Database

PostgreSQL 16 with Row-Level Security, partitioned by tenant + date. Redis 7 for cache/queues.

Security

MFA, JWT + refresh rotation, RLS, AES-256 encryption, OWASP ZAP, Semgrep, Trivy, Gitleaks.

Testing

Vitest (unit), Testcontainers (integration), Supertest (e2e), Playwright (UI across 5 browsers).

Queue

BullMQ for ingestion workers, classification jobs, balance recalc, notifications, exports.

Deployment

Docker, Kubernetes-ready. GitHub Actions CI. CDN via CloudFlare. S3 for file storage.

Monitoring

Grafana + Prometheus. Structured logging. Error tracking. Uptime monitoring with alerting.
