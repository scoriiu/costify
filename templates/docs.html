Introduction to Costify

Costify is the financial operating system for Romanian businesses. This guide will help you understand the core concepts and get started quickly.

What is Costify?

Costify is a multi-tenant financial platform that serves two audiences from the same data: the accountant who processes financial records, and the entrepreneur who needs to understand and control their business.

Unlike traditional accounting software (Saga, SmartBill) that focuses only on compliance, Costify adds a management intelligence layer: real-time dashboards, budgets, forecasts, cost classification, and treasury management.

ⓘ

If you're migrating from Saga, check our Saga Migration Guide for a step-by-step walkthrough of importing your chart of accounts and historical data.

Core Concepts

Every piece of financial data in Costify flows through the processing pipeline:

StageWhat happensAudit
IngestRaw data fetched from bank sources (FGO, Open Banking, CSV)Raw payload hash stored
NormalizeParsed, mapped to unified schema, deduplicatedBefore/after diff recorded
ClassifyRule engine assigns category, vertical, owner, confidence scoreMatched rule + score logged
JournalWritten to immutable ledger with sequential numberingEntry + allocations recorded
BalancesAccount balances recalculated, snapshots storedPrevious/new balance delta

Quick Example

Here's how you'd connect a bank source and trigger the first sync using the API:

typescript
// Create a bank source for FGO.ro
const source = await costify.sources.create({
  name: 'BT Business via FGO',
  type: 'fgo',
  bankAccountId: account.id,
  config: {
    fgoAccountId: 'acc_12345',
  },
  syncFrequency: 'hourly',
})

// Trigger initial sync
const syncResult = await costify.sources.sync(source.id)
console.log(syncResult.transactionsImported) // 247
✔

Tip: Each bank account can have exactly one active source (FGO or direct bank, never both) to prevent duplicate transactions.

Multi-Tenancy

Costify is built for accountants managing hundreds or thousands of client companies. Each company is a fully isolated tenant with its own data, users, and configuration.

typescript
// Switch tenant context
const client = costify.tenant('tenant_abc123')

// All subsequent calls are scoped to this tenant
const transactions = await client.transactions.list({
  dateFrom: '2026-01-01',
  status: 'unclassified',
  limit: 50,
})
⚠

Security: Row-Level Security (RLS) is enforced at the database level. Even if application code has a bug, the database will never return data from another tenant.
